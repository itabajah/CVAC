# Templates

Pre-built resume templates that serve as starting points for your own resumes.

## Usage

1. Copy any template folder to `resumes/`
2. Rename it to something meaningful (e.g., `my-resume`)
3. Edit the HTML/CSS to match your information

## Available Templates

- **example1/** - Two-column layout with dark sidebar and accent colors
- **example2/** - Bold typography with timeline and skill bars

## Notes

- Templates are read-only and updated automatically via git pull
- Never edit templates directly - copy to `resumes/` first
- Templates appear in the UI dropdown as `templates/example1`, `templates/example2`, etc.
